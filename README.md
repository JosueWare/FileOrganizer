Este script faz que os arquivos todos bagunçados em uma determinada pasta sejam separados em cada pastas de tipos de arquivos, seja Fotos, Vídeos, GIFs, Documentos e etc. Utilizando o PowerShell para automações.

Antes de começar usar siga os passo a passo.

1. Execute primeiramente o arquivo `Run.bat` pois a pasta "Source" não existe.
2. Pegue seus arquivos bagunçados e jogue na pasta "Source" e prossiga em executar o arquivo `Run.bat` novamente (Se já estiver executado.)
3. No Terminal haverá uma questão se você deseja iniciar, digite "S" de 'Sim'.
4. Seus arquivos serão tudo organizados em cada pastas pessoais do sistema, todas nomeadas como "**FileOrganizer**".

Observações:

- Recomendado você utilizar o PowerShell em vez do Windows PowerShell (O legado do Windows) pois pode haver erros de caracteres.

O sistema possui suporte a diversas extensões de arquivos, incluindo: 

- Imagens: PNG, JPG, JPEG
- Vídeos: MP4
- Documentos: TXT
