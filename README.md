Este script faz que os arquivos todos bagunçados em uma determinada pasta sejam separados em cada pastas de tipos de arquivos, seja Fotos, Vídeos, GIFs, Documentos e etc. Utilizando o PowerShell para automações.

Antes de começar usar siga os passo a passo.

1. Execute primeiramente o arquivo `Run.bat` pois a pasta "Source" não existe e ela será criada automaticamente.
2. Pegue seus arquivos bagunçados e jogue na pasta "Source".
3. Com o Terminal aberto haverá uma questão se você deseja iniciar, digite "S".
4. Seus arquivos serão tudo organizados em cada pastas pessoais do sistema, todas nomeadas como "**FileOrganizer**".

Observações:

- Recomendado você utilizar o PowerShell 7 em vez do Windows PowerShell (O legado do Windows) pois pode haver erros de caracteres e outros recursos não disponíveis.

O Script possui suporte a diversas extensões de arquivos, incluindo: 

- Imagens: PNG, JPG, JPEG
- Vídeos: MP4
- Documentos: TXT
